# Project Hub

A simple project showcase app built with React, TypeScript, Vite, and Tailwind CSS.

## Live Demo

`https://ybenbrai.github.io/project_hub/`

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
